# Project Overview

## Summary:
I am developing a dashboard for a real estate company, this repo is for its dashboard and there is another seperated repo for the website.

## Features: 
The dashboard needs to provide the simple basic features like:

1. Ability to login ( no need for the sign in or create account option )
2. Sidebar showing navigation to routes like ( main , properties , profile )
3. The Main page will for now will be empty ( i will decide later )
4. The Properties page will list some recently added properties, ability to select a single property and view it, then option to update it and delete it. plus a search bar with filters.
5. The Profile section will have a form to update the company fields like name, bio, logo and more ( more details are in the data-model.md file )

## Tech Stack:

**Frontend**: Nuxt + Tailwind + Nuxt UI + Pinia
**Unit Testing**: Vitest
**Backend**: Firebase (firestore, storage, hosting, Emulator)
**Error Handeling**: Sentry

more details will be provided in the /context/architecture.md & data-model.md files
